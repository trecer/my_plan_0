## Задание: Бот-анкета с админкой

Напишем бота, который использует всё изученное:

**Функционал:**
1. Пользователь заполняет анкету (имя → возраст → город) через FSM
2. Данные сохраняются (пока в памяти)
3. Админ может посмотреть все анкеты
4. Inline-кнопки для подтверждения/отмены

**Что затронем:**
| Концепция | Где в боте |
|-----------|-----------|
| Router | user_router, admin_router |
| FSM | Сбор анкеты |
| Кастомный фильтр | IsAdmin |
| Middleware | Логирование |
| Inline клавиатуры | Подтверждение |
| Структура проекта | Разделение на файлы |
